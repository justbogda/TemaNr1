	
	Web crawler este un motor de cautare care navigheaza sistematic. 
	Scopul lui este de a indexa si descarca paginile web, astfel incat va putea efectua urmatoarele
operatiuni: filtrare dupa tip, cautarea dupa cuvinte cheie, limita dimensiune, creare sitemap.

	Aplicatia primeste un fisier cu URL-uri si un fisier de configurare in care sunt precizate numarul de thread-uri,
delay, nivel de adancime, numele directorului unde se afla toate informatiile despre paginile web.
